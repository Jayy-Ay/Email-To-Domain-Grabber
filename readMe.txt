How to use my DomainGrabber Step-By-Step:
-----------------------------------------------
Requirements:
- A High-Level Programming IDE
- Pandas Library
  - Write "pip install pandas" on your powershell terminal to install
- a'Microsoft Excel'/'Google Sheets'/etc. Ensure that it has a column named 'Email'
-----------------------------------------------
Steps On How To Use:

1. Turn your 'Microsoft Excel'/'Google Sheets'/etc. into '.csv' form.
2. Copy the directory path of the '.csv' file 
3. Run the Program
4. Paste the output in the python terminal

Note: Rerun the program to use it again
